# STORY: V080-DEV-003 — Pipeline 2-Image Carousel Deselection

## Status: Done
## Assignee: developer
## Brief: V080-POLISH

## Summary
When a carousel has exactly 2 images, the user cannot deselect either one because `canRejectMoreInCarousel` requires `nonRejectedCount > CAROUSEL_MIN_IMAGES (2)`. This locks the user — they can't reduce to 1 image.

## Acceptance Criteria
- [x] User CAN reject one image from a 2-image carousel
- [x] When reduced to 1 image, the carousel degrades to a single post (not a broken 1-image carousel)
- [x] The single remaining image is posted as a standalone image post
- [x] No error prompts or broken UI states
- [x] Whole-carousel reject still works (rejects all images)

## Resolution
`CAROUSEL_MIN_IMAGES` dropped from 2 → 1 in
`lib/carousel-degrade-guard.ts`. A 2→1 per-image reject is now
permitted; the survivor is collapsed into a single-image post by
`groupApprovalPosts` (any 1-sibling carousel group becomes a single
card), and the `/api/social/post` route treats `mediaUrls.length === 1`
as a single-image post (only fans out to Instagram's CAROUSEL
container when `igMediaUrls.length > 1`). Only the very last image is
still locked behind the explicit "Reject carousel" button so killing
the whole post stays deliberate. Covered by
`tests/lib/carousel-degrade-guard.test.ts` plus the integration
suites `carousel-approval-flow.test.tsx` and `carousel-ghost.test.ts`.

## Technical Notes
- lib/carousel-degrade-guard.ts: `CAROUSEL_MIN_IMAGES = 2`, `canRejectMoreInCarousel` returns `nonRejectedCount > 2`
- When user rejects 1 of 2 images → remaining image should be converted to single post
- The auto-poster rejects 1-image carousels (per the comment in degrade-guard)
- Solution: allow the rejection, then handle the degrade — either:
  a) Convert the ScheduledPost from carousel to single image, or
  b) Show a confirmation: "This will post as a single image instead of a carousel"
- Check CarouselApprovalCard.tsx line 177 for the reject handler
- Check lib/carousel-ghost.ts for ghost/degrade logic
