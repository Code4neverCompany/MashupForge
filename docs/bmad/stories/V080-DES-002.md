# STORY: V080-DES-002 — Post Ready Countdown Timer

## Status: Done
## Assignee: designer
## Brief: V080-POLISH

## Summary
Add a visible countdown/timer badge on PostReady cards showing time remaining until the post goes out. The sorting is already correct (soonest first), but there's no visual indicator of when each post is scheduled.

## Acceptance Criteria
- [ ] Each PostReady card shows a countdown badge (e.g., "in 2h 15m", "in 3 days", "overdue")
- [ ] Countdown updates in real-time (or at least every minute)
- [ ] Overdue posts are visually distinct (red/amber badge)
- [ ] Unscheduled posts show "Not scheduled" or similar
- [ ] Badge doesn't clutter the card — compact, single-line

## Technical Notes
- PostReadyCard.tsx and PostReadyCarouselCard.tsx in components/postready/
- The scheduled time comes from `settings.scheduledPosts` — each has `date` and `time` fields
- Use a simple `useEffect` + `setInterval` for live countdown
- Format: "in 2h 15m", "in 1d 3h", "in 45m", "overdue by 10m"
- Style: small pill/badge in the card corner, use brand gold (#c5a062) for active, red for overdue
